# Job Tracker - Web Dashboard

A React-based web dashboard for managing job applications, sharing the same backend and logic as the mobile app.

## Features

- **Board / List View**: Organize jobs visually.
- **Drag & Drop**: Powered by `@dnd-kit` for sorting jobs.
- **Shared Logic**: Uses `@job-tracking/shared` for consistent types and utilities.

## Setup

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```
   Access at `http://localhost:5173`.

## Tech Stack

- **React** (Vite)
- **TypeScript**
- **@dnd-kit**: Drag and drop interactions
- **TailwindCSS** (styled components)
- **Lucide React**: Icons
